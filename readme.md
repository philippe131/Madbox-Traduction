# Madbox-Traduction

Translation web app using NodeJs / ExpressJs / React / MongoDb

## Installation

To install the dependencies
```
npm install
cd client && npm install
```

## Test

Run tests
```
jasmine
```

## run the app

Run nodejs
```
npm start
```

Run React app
```
npm run client
```
